# Maths Circle Tools

Stuff written for the Maths Circle (Club) that we run for Y4-Y6 children in London.

Most of these tools are vanilla JS solutions, vibe-coded without actually looking into the code. Each file serves a single purpose: to demonstrate something in mathematics or to render something which we then transfer onto printed cards with individual challenges.
