# Maths Circle Tools

Stuff written for the Maths Circle (Club) that we run for Y4-Y6 children in London.

Most of these tools are vanilla JS solutions, vibe-coded without actually looking into the code, and serve a single purpose: to demonstrate something in mathematics or to render something that we then transfer onto printed cards with individual challenges.

**Try the tools:** [maths-circle-tools.dm9.dev](https://maths-circle-tools.dm9.dev/)
